# Online-Shopping-system-using-Tkinter-and-SQL
Implementation of the system is by using MySQL and a python graphics library called Tkinter.
The 3 main modules are:
## USER:
Tasks:
1. Login
2. Register
3. Purchase an item
## ADMIN:
Tasks:

1. Manage user
2. Manage moderator
3. Manage products
4. Manage orders

## MODERATOR:
Tasks:
1. Manage products
2. Manage users
3. Manage orders
